内存与存储成本在近一年大幅上涨。受此影响，Usolia 使用的供应商也在不断涨价

Usolia 此前使用的游戏服务器服务商 Hetzner 对 Usolia 曾使用的 AX102 涨价了14%（€109 -> €124），并在今年6月份将新购该服务器的价格提高了一倍（€124 -> €259 每月，外加€129的一次性费用）

目前使用的服务商 OVHcloud 也在5月初进行了涨价，并于本文撰写不久前[宣布了进一步涨价](https://x.com/olesovhcom/status/2069848711442694163) (@olesovhcom, X)，对已有客户预计在本年10月初生效

Usolia 运营的游戏服务器需要大量内存，因此也对价格浮动的影响十分敏感。目前硬件市场供应短缺尚未出现缓和迹象，在性能水准不变的条件下，每月运营费用也将继续提高。
