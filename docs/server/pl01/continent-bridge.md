## 概念

大陆桥分散在世界各地，其通常包含一个信标用于标识，像这样：

![](https://assets-docs.usolia.net/docs.usolia.net/images/general/cbridge.png)

其目前在以下地点设有节点：

### Earth

 - 北美洲（纽约）
 - 欧洲（伦敦）
 - 非洲（开普敦）
 - 亚洲（北京）
 - 大洋洲（悉尼）
 - 南美洲（圣保罗）

### Lavender

- 出生点
